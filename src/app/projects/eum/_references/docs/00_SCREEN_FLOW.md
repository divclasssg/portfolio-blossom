# 이음 프로토타입 v2.0 — 화면 흐름 정의

> 스토리: 6년간 원인을 못 찾던 윤서진이 서현내과의원에서 이음 앱을 설치하고, 3개월 만에 AI가 자율신경계 불균형 패턴을 발견하여 신경과 타과의뢰를 받기까지.

## 스토리 타임라인

```
2020     역류성 식도염 발병 (LA-A) → 서현내과의원 김도현
  ↓      6년간 반복 방문, 내시경 4회 정상(LA-N), 원인 불명
2021     스트레스·수면 문제 악화 (정식 진단 없음)
─────── ✦ 이음 앱 도입 ✦ ──────────────────────
2025-12  서현내과의원 방문 → 이음 앱 설치 ← [온보딩 시점]
  ↓      ses_001: 첫 진료 (기능성 역류 가능성 첫 언급)
  ↓      ses_002: 불안-소화기 상호작용 패턴 감지
  ↓      ses_003: 수면↔증상 상관 확인 (2026-02-10)
2026-02  환자 HOME ← [홈 화면 시점: ses_004 이전, 마지막 방문 ses_003]
  ↓      ses_004: AI 6년치 종합 분석 ← [의사 대시보드 시점]
  ↓      → "자율신경계 불균형 가능성" 제시
  ↓      → 김도현, 분당신경과의원 (이음 연계) 타과의뢰 결정
```

## 네비게이션 흐름

### 환자 앱

```
P-000 면책
  → P-001 로그인/회원가입
      ├─ [로그인] → P-015 (GPS 확인)
      └─ [회원가입] → P-002 → P-003 → P-004 → P-005 → P-006
          → P-007 생체인증
              ├─ [설정] → P-008 (Face ID) → P-009
              └─ [나중에] → P-009
          → P-009 마이데이터
              ├─ [연동] → P-010 → P-011
              └─ [나중에] → P-011
          → P-011 웨어러블
              ├─ [Apple] → P-012 → P-013
              └─ [완료/나중에] → P-013
          → P-013 건강정보 입력 → P-014 가입 완료
              → P-015 병원 GPS (서현내과의원)
                  ├─ [맞아요] → P-017 체크인
                  └─ [다른 병원] → P-016 → P-017
              → P-017 체크인 (서현내과의원 김도현)
                  → P-018 HOME
                      ├─ [증상 기록] → P-019 → P-018
                      ├─ [지난 진료 결과 상세] → P-020 → P-018
                      └─ [탭: 진료요약] → P-020
```

### 의사 프로그램 (김도현)

```
D-000 대시보드 (ses_004 진료중)
  ├─ [데이터 보기] → D-F12 타임라인
  │     └─ [닫기] → D-000
  └─ [진료 종료] → D-001 결과 확인
        └─ [전송] → 완료 (환자 P-020으로 결과 전달)
```
