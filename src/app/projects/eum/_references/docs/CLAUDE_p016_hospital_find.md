## 양쪽 일치 결정사항 (5건)

| 항목 | 값 |
|---|---|
| 뒤로가기 | ← P-015 |
| 근처 병원 리스트 | GPS 기반 거리순 정렬 |
| 카드 구성 | 병원명 + 거리 + 주소 |

## 포트폴리오 데모 제외 항목 (3건)

| 항목 | 스펙 내용 | 제외 근거 |
|---|---|---|
| GPS 비활성 에러 | "위치 권한이 필요해요" | 실제 GPS 미사용, 하드코딩 데이터 |
| 근처 병원 없음 에러 | "근처에 Eum 병원이 없습니다" | 항상 결과 존재하는 데모 시나리오 |
| 3회 실패 잠금 | 카운트다운 타이머 | 백엔드 없는 데모, 인라인 에러면 충분 |
